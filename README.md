# CircuitCrate – Electronics Store Website

CircuitCrate is a simple **web-based electronics store project** built using **HTML, CSS, and JavaScript**.  
It provides a basic interface for users to log in, browse electronics products, search for items, and view product details.

This project is designed as a **front-end web development project** demonstrating UI design, navigation between pages, and basic JavaScript functionality.

---

## 📌 Features

- 🔐 Simple Login Page
- 🛍️ Electronics Store Page with multiple products
- 🔎 Product Search functionality
- 📄 Product Details page
- ℹ️ About Us page with contact information
- 🚪 Logout functionality
- 🎨 Responsive layout using CSS Grid
- ⚡ Dynamic behavior using JavaScript

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

## ⚙️ How to Run the Project

1. Download or clone the repository:

```bash
git clone https://github.com/your-username/circuitcrate-store.git
```

2. Open the project folder.

3. Run the website by opening:

```
index.html
```

in your browser.

## 🔎 Search Feature

The store includes a **live search feature** that allows users to filter products instantly by typing in the search bar.

Example:

```
Search: Arduino
```

This will display only Arduino-related products.
